# Term-tree

Term-tree is a basic terminal-like startpage with tree structure.

## Authors

Mehedi Hasan (@0xMehedi) and the [contributors](https://github.com/0xMehedi/term-tree/graphs/contributors).

## Features

- Supports Firefox and Chromium

- Optimized for Speed

- Cross Platform

- Mobile Friendly
